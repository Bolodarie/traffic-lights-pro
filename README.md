# Traffic Light Detection Framework

Este repositório contém a implementação modular para detecção de semáforos utilizando Deep Learning.

## Estrutura do Estágio
1. **Etapa 1:** Estudo Teórico (Notebooks)
2. **Etapa 2:** Dados (src/data)
3. **Etapa 3:** Desenvolvimento (src/models)
4. **Etapa 4:** Testes (src/engine)

## Como rodar
1. Instale as dependências: `pip install -r requirements.txt`
2. Execute o treino: `python main.py --model resnet18`
